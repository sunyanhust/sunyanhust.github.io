服务器使用screen后台运行程序

2020-04-01 1 min read
  1. 安装screen：yum install -y screen
  2. 确认远程服务器是否安装screen：screen -v
  3. 启动screen 会话：screen -S train，会话名字为train。这个时候会切换到新的会话终端，在里面运行我们想要的程序。
  4. 切换返回： ctrl+a+d
  5. 查看运行中的screen会话：screen -ls
  6. 看到对应的id后，重新登录会话,，这里假设id为774：screen -r 774，如果登录不上，使用screen -D -r 774 先踢掉前一用户，再登陆。
